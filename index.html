FTPM

font package manager

FTPM is a Font helper built on top of Google Web Fonts and Node.js. Allows you to manage system fonts, generate web font files, CSS from Google Web Fonts and CSS using data-uri embeds. It requires NPM to setup:
npm install -g ftpm with Root privileges.

System font

ftpm install “droid sans”

ftpm uninstall magra

ftpm local

Web font file

ftpm web magra

ftpm web “droid sans”

CSS font

ftpm css magra

ftpm datauri “droid sans”

ftpm css magra -s
